AplJAX Jackson Serialization Test

This is a basic unit test that will create a full serialized directory of a JAX-RS application as JSON using Jackson.

- Checkout dbennett455/apljax from github.com.
- Make sure apljax is in your classpath
- Put your JAX-RS application in the classpath
- Run TestJackson.java as a Junit test

The JAX-RS application full API will be output to formatted JSON
